# Microsserviço de Autenticação (Simples)

## Tecnologias
- Node.js
- Express
- PostgreSQL
- SQL puro (sem ORM)

## Como usar
```bash
npm install
npm run dev
```

## Endpoints
- POST /api/v1/auth/signup
- POST /api/v1/auth/login
- POST /api/v1/auth/recuperar-senha
- POST /api/v1/auth/logout
- GET  /api/v1/auth/me
